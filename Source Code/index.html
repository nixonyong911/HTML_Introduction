What is XMLHttpRequest

XMLHttpRequest, also known as XHR, is an API which provides clients the ability to transfer data between client and server. XHR allows web browser to retrieve data from URL without performing refresh. With this function, web page is able to update only part of it without interrupting the user's' current task. Although it is mainly AJAX programming, it is also compatible with VBscript, JavaScript and many other different web browsing scripting languages to transfer and edit XML data. Lastly, it can be used to fetch data in different formats such as JSON or plain text.

Example of XMLHttpRequest in real world

  • Gmail
  • New email were updated while user are composing, replying or forwarding emails.
  • Facebook
  • New feeds are automatically updated while user are browsing their home page
  • Youtube
  • Playlist is automatically generated and new recommended playlist were added as user is watching video.
