Timer

The timer is set to 60 second, and after 60 second it will automatically submit the page.

This is achieved by initialing timer function when the page is loaded. When the timer ends, it will call another function which will submit the form automatically after alerting user that their time is up.

Showing timer and prevent clashing

Since there is a timer on the page, user should know the time left, therefore the page has implemented a timer on top and will show user the time left.

It is achieved by constantly updating the text every second.

Another problem is when both JavaScript has window.onload, which will rush since both have the same name.

This can be solved by using JQuery instead of pure JavaScript, thus both function will look different and will be able to load like normal.
