Rotating and dropdown menu

Rotating on hover effect can be achieved by using transition and transformation css style

Transition indicate how long the animation will be, and transformation indicate what kind of animation, in this case, its a rotation animation.

Dropdown menu can be achieved by setting 1st hide the content (display:none) and as the mouse hover over, make it display as block(display:block) .

Next we have to set the position to be absolute for the dropdown content, this prevent the page to move.

Parallax Scrolling Effect

Create an illusion that 2 object are moving at different speed, example when user scrolls the webpage, this sunrise image remain still. Thus creating parallax scroll effect.

There are 3 part to get this effect works

  1. Create the section for the image and set the background of the section.
  2. Set the background image attachment(background-attachment) to be fixed.
  3. After setting the image to be still, next we have to make sure the image is cover and no repeating, therefore (background-repeat and background-size) is used.
